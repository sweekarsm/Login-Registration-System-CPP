# Login and Registration System in C++

A console-based authentication system built using C++ and SQLite.

## Features
- User Registration
- User Login
- Password Hashing
- SQLite Database Integration
- Prepared Statements
- SQL Injection Prevention
- Hidden Password Input

## Technologies Used
- C++
- SQLite
- Visual Studio

## Future Improvements
- GUI Interface
- Better Password Encryption
- OOP Refactoring
- Forgot Password Feature
